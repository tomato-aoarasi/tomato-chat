# Tomato AI Chat  
Tomato的Nonebot1的AI Chat功能,你需要本地有ollama且安装了大模型  
写着玩开心就可以(  
py代码写的烂,轻喷(

## 安装三方包  
```bash
$ pip install -r requirement.txt
```

## 如何运行大模型?  
需要本地安装[ollama](https://ollama.com/)  
具体看ollama官方教程  

## 配置bot
全部配置内容都在bot_config.py中  
默认是不给全部群开启chat的,请手动配置allow_groups添加允许  

## 启动nonebot1后台  
```bash
$ python3 main.py
```
